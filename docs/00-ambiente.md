# Ambiente do Laboratório

## 🖥️ Virtualização
Ambiente criado utilizando VirtualBox.

## 🧠 Servidor
- Sistema: Windows Server
- Função: Controlador de Domínio (AD DS)
- Serviços:
  - Active Directory
  - DNS

## 💻 Cliente
- Sistema: Windows 10
- Integrado ao domínio

## 🌐 Rede
- Comunicação entre servidor e cliente
- Configuração de IP e DNS manual

## 🎯 Objetivo
Simular um ambiente corporativo com Active Directory funcional.
